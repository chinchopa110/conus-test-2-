# conus.py принцип работы
## 1: area(r,h)
### Параметры функции
- h - высота конуса.
- r - радиус конуса.
### Возвращает:
- площадь поверхности конуса 
### Пример:
- input-> 5 4
- area(5, 4)
- return pi * 5 * (4 + ((4**2) + (5**2))**0.5)
- output -> 179.1197053653833
## 2: sum(r,h)
### Параметры функции
- h - высота конуса.
- r - радиус конуса.
### Возвращает:
- сумму радиуса и высоты 
### Пример:
- input-> 5 4
- area(5, 4)
- return 5 + 4
- output -> 9
